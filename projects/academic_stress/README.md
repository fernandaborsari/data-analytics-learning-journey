
# Academic Stress ETL Project

## Overview

This project demonstrates a complete ETL pipeline built using Python and pandas.

The objective is to extract, clean, transform, and prepare data for analysis while applying version control best practices.

Developed using Google Colab and version-controlled with GitHub.

---

## Dataset

Raw data:  
data/raw/academic_stress.csv

Processed data:  
data/processed/academic_stress_clean.csv

---

## ETL Process

Implemented in:  
python/etl_study.ipynb

The ETL workflow includes:

- Extracting the raw dataset
- Cleaning and standardizing data
- Handling missing values
- Transforming columns for analysis
- Exporting a processed dataset ready for SQL analysis
